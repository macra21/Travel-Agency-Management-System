# Travel Agency Management System ✈️

> A C++ and Qt desktop application for managing holiday offers, built strictly on Domain-Driven Design (DDD) principles. It ensures high reliability and performance through comprehensive automated testing with high code coverage, leak-free memory management verified via Valgrind, and execution optimizations guided by profiling tools.

## Features
* **Offer Management:** Add, update, delete, and securely store holiday packages and travel offers.
* **Smart Filtering:** Quickly search and filter destinations based on price, location, or availability.
* **Interactive GUI:** User-friendly and responsive interface powered by the Qt Framework.
* **Decoupled Logic:** Clean separation between the business domain and the presentation layer.

## Technologies & Architecture
* **Core Language:** C++
* **GUI Framework:** Qt
* **Architecture:** Domain-Driven Design (DDD), Layered Architecture
* **Principles:** Object-Oriented Programming (OOP), SOLID

## Testing & Performance
Quality assurance and system stability were top priorities for this project:
* **Memory Management:** Rigorously tested with **Valgrind** to ensure zero memory leaks.
* **Automated Testing:** Core domain functionalities are covered by unit tests, achieving **high code coverage**.
* **Performance Optimization:** System execution times and bottlenecks were analyzed and optimized using advanced **profiling tools**.
